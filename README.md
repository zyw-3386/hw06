# hw06"大家讲 AI"短视频制作

## 视频选题简介
本次短视频作业的主题为**具身智能**，旨在用通俗易懂的方式，向观众介绍具身智能的核心概念、技术特点、典型应用场景与发展意义。

视频时长约3分钟，内容分为以下几个部分：
1. **开场引入**：对比传统AI与具身智能的区别，点明具身智能是AI发展的重要方向。
2. **核心定义**：解释具身智能“有身体的人工智能”这一核心概念，说明其感知、行动、互动的能力。
3. **关键特点**：以“倒水”为例，说明具身智能“知行合一”的特点，以及与真实环境互动的重要性。
4. **应用场景**：介绍家庭服务机器人、工业机械臂、巡检机器人等典型应用，展现技术价值。
5. **未来展望**：简述具身智能的发展趋势，说明其将推动AI从虚拟走向物理世界的意义。
